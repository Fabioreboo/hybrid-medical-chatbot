# Directives

This directory contains Standard Operating Procedures (SOPs) written in Markdown.

These files define the goals, inputs, tools/scripts to use, outputs, and edge cases for the agent.
Instead of the agent doing everything itself, it reads these directives to orchestrate and execute deterministic Python scripts in the `../execution/` directory.

## Operating Principles
- Define clear inputs and outputs
- Specify edge cases and how to handle them
- Keep instructions in natural language
